# 🇮🇱 Israel Civilian Readiness Guide

A comprehensive field guide and checklist system for civilian preparedness during wartime in Israel.

## ⚠️ DISCLAIMER

**This is NOT an official government resource.** This guide is community-authored by Daniel Rosehill with AI assistance. It is based on publicly available guidance from the Israel Home Front Command (Pikud HaOref / פיקוד העורף) but carries **no warranty of any kind**.

**Always follow official instructions** from HFC, the IDF, Israel Police, and local authorities. In the event of any conflict between this document and official guidance, **follow official guidance**.

**Official source:** [oref.org.il](https://www.oref.org.il) | **HFC Hotline:** 104

### Grounding & Verification

This guide was systematically cross-referenced against official HFC guidance pages from the English-language National Emergency Portal (oref.org.il), archived on **20 March 2026**. Source HTML files are preserved in `sources/hfc-official/` for transparency and verification.

## What's In This Guide

- **Quick Smoke Test (PAWS BED)** — 7-item rapid readiness check with mnemonic
- **Master Checklist** — Full readiness verification (tech, home, go bag, people, personal effects, situational awareness)
- **Situational Checklists** — Daytime posture, after shelter reset, before showering, before bed, before leaving home
- **Guidance Notes** — Go bag selection, shelter behaviour, moving safely, night alarms, alert fatigue, protracted conflict wellness, caring for vulnerable neighbours, public shelters, OPSEC, preliminary guidelines, infiltration protocol
- **Appendices** — HFC app config, emergency numbers, mamad inspection, protected space decision tree, Hebrew glossary, emergency radio frequencies
