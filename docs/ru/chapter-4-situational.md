# Глава 4: Ситуационные чек-листы

## 4.1 Дневной режим дома

*Быстрая проверка статуса, находясь дома днём. Выполняйте после пробуждения и повторяйте после любого перерыва.*

| | ПУНКТ | ПРОВЕРКА |
|---|------|----------|
| 📱 | **Телефон** ⚠️ | ☐ **ПРИ СЕБЕ** *Или на расстоянии вытянутой руки* · ☐ **ЗАРЯЖЕН ИЛИ ЗАРЯЖАЕТСЯ** · ☐ **ПРИЛОЖЕНИЕ HFC ЗАПУЩЕНО** |
| 👕 | **Одет(а)** ⚠️ | ☐ **ПОЛНОСТЬЮ ОДЕТ(А)** · ☐ **ОБУВЬ С ЗАКРЫТЫМ НОСКОМ НАДЕТА** *Не тапки, не босиком* |
| 🔑 | **Ключи** ⚠️ | ☐ **ПРИ СЕБЕ ИЛИ В ДВЕРИ / У ДВЕРИ** |
| 🎒 | **Тревожная сумка** ⚠️ | ☐ **У ДВЕРИ** · ☐ **ЗАСТЁГНУТА** |
| 🚪 | **Путь эвакуации** ⚠️ | ☐ **КОРИДОР СВОБОДЕН** · ☐ **ВХОДНАЯ ДВЕРЬ БЫСТРО ОТКРЫВАЕТСЯ** |
| 💻 | **Оповещение в браузере** | ☐ **РАСШИРЕНИЕ RED ALERT АКТИВНО** *Если работаете за компьютером* |

!!! tip "Работа из дома во время войны"
    - Оставайтесь одетыми и обутыми — не переодевайтесь в пижаму «просто за столом»
    - Наушники: используйте только одно ухо или держите громкость на уровне, позволяющем слышать сирену
    - Сохраняйте работу часто — возможно, придётся бросить всё на полуслове
    - В начале любого звонка: *«Я нахожусь в зоне активных оповещений. Возможно, мне придётся срочно уйти.»*

## 4.2 После возвращения из укрытия (сброс)

*Выполняйте сразу после отбоя тревоги. Восстановите полную готовность до следующего оповещения.*

| | ПУНКТ | ПРОВЕРКА |
|---|------|----------|
| 🎒 | **Тревожная сумка** ⚠️ | ☐ **ВЕРНУЛАСЬ К ДВЕРИ** · ☐ **ЗАСТЁГНУТА ЗАНОВО** *Что-нибудь выпало? Положите обратно сейчас* |
| 📱 | **Телефон** ⚠️ | ☐ **НА ЗАРЯДКЕ** *Если разрядился* · ☐ **ПРИЛОЖЕНИЕ HFC ПРОДОЛЖАЕТ РАБОТАТЬ** |
| | **Внешний аккумулятор** | ☐ **НА ЗАРЯДКЕ** *Мог использоваться в укрытии* |
| 💧 | **Вода** | ☐ **ЗАПАС ПОПОЛНЕН** *Если пили из аварийного запаса* |
| 👕 | **Одежда и обувь** | ☐ **НА МЕСТЕ** *У кровати (ночь) или на себе (день)* |
| | **Фонарь** | ☐ **НА МЕСТЕ** *Тумбочка или тревожная сумка* |
| 👥 | **Иждивенцы** ⚠️ | ☐ **ВСЕ НА МЕСТЕ И В БЕЗОПАСНОСТИ** |

!!! note "Самочувствие"
    **Самообслуживание — сделайте это СЕЙЧАС, а не потом.** Вы не знаете, когда будет следующая тревога.

    - [ ] **ПОЕЛ** — *Стресс подавляет аппетит — ешьте всё равно. Вам нужна энергия.*
    - [ ] **ПОПИЛ**
    - [ ] **ТУАЛЕТ — СДЕЛАНО**
    - [ ] **ПОМЫЛСЯ** — *Если время и ситуация позволяют*
    - [ ] **УХОЖЕН** — *Моральный дух и ощущение нормальности важны*
    - [ ] **ОТДОХНУЛ** — *Когда возможно*

    **Дети:**

    - [ ] **НАКОРМЛЕНЫ** · **НАПОЕНЫ** · **СХОДИЛИ В ТУАЛЕТ / ПЕРЕОДЕТЫ**
    - [ ] **ПОМЫТЫ** *Если возможно* · **УЛОЖЕНЫ** *Положите спать, если позволяет время*
