# الفصل الثالث: قائمة التحقق الرئيسية

## 3.1 الأنظمة التقنية والتنبيهات

| | العنصر | التحقق |
|---|------|-------|
| 📱 | **الهاتف الذكي** ⚠️ | ☐ **معك شخصياً** *أو موقعه معروف* · ☐ **مستوى البطارية — جيد** *أو يشحن* |
| 📡 | **إعدادات الهاتف الذكي** ⚠️ | ☐ **مُشغّل** *وضع عدم الإزعاج مقبول؛ وضع الطيران غير مقبول* · ☐ **خدمات الموقع — مُفعّلة** · تطبيق قيادة الجبهة الداخلية: ☐ **مُثبّت** · ☐ **مُعدّ** *انظر الملحق أ* · ☐ **يعمل** · ☐ **منطقة التنبيه الصحيحة — مُتحقق منها** · تنبيهات الطوارئ اللاسلكية: ☐ **مُفعّلة** *تأكد من تفعيل تنبيهات التهديدات "القصوى"* |
| 💻 | **الحاسوب** | تطبيق الإنذار الأحمر للمتصفح: ☐ **نشط** · ☐ **الموقع الصحيح مُعيّن** · ☐ **تم الاختبار والتحقق من عمله** |

## 3.2 البيئة المنزلية

| | العنصر | التحقق |
|---|------|-------|
| 💧 | **الماء** ⚠️ | ☐ **مخزون 72 ساعة — جاهز** *احسب 3 لتر/شخص/يوم. أعد التزويد عند الحاجة* |
| 🍽 | **الطعام** ⚠️ | ☐ **المؤن مُخزّنة** *انظر الملحق ط لمؤن الطوارئ الإسرائيلية* |
| | **المصباح** | ☐ **في متناول اليد** *يدوي أو بطارية مع احتياطي* |
| 🚪 | **الممر** ⚠️ | ☐ **خالٍ** *بدون عوائق* |
| 🧯 | **طفاية الحريق** | ☐ **في متناول اليد** · ☐ **كاشف الدخان — مُثبّت ومُختبر** |
| 🚪 | **مسارات الهروب** | ☐ **المخرج الرئيسي مُحدد** · ☐ **المخرج الثانوي مُحدد** *باب خلفي، شرفة، سلالم طوارئ* · ☐ **المسارات خالية من العوائق** |
| 👶 | **حمّالة الطفل** | ☐ **بجانب الباب الأمامي** |
| 🛡 | **المم"د** | ☐ **مُجهّز** *انظر الملحق ج — فحص المم"د* |

*تثبيت الأثاث الثقيل، ومفاتيح إغلاق الغاز/الكهرباء، وسلامة الحريق — انظر الملحق ز: السلامة المنزلية قبل الطوارئ. هذه مهام تحضيرية وليست فحوصات نشطة في زمن الحرب.*

## 3.3 حقيبة الطوارئ

*انظر الفصل 5 §5.1 والملحق ح لتوجيهات مفصلة حول حقيبة الطوارئ.*

| العنصر | التحقق |
|------|-------|
| **مُجهّزة** ⚠️ | ☐ **بجانب الباب** · ☐ **مغلقة / محكمة الإغلاق** |
| **المحتويات (أساسية)** ⚠️ | ☐ **شاحن الهاتف والكابل — معبأ** · ☐ **بنك الطاقة — معبأ** *مستوى الشحن مُتحقق أو يشحن* · الأدوية: ☐ **الأدوية الأساسية — معبأة** · ☐ **المخزون مُراجع ومُجدّد** · ☐ **المصباح — بالداخل ومشحون** · ☐ **كمامة N95 — معبأة** *واحدة لكل شخص* |
| **المحتويات (إضافية)** | ☐ **صافرة طوارئ — معبأة** · ☐ **راديو AM/FM — معبأ** *بطارية أو شحن جيد* · ☐ **حقيبة إسعافات أولية — معبأة ومزوّدة** · ☐ **راوتر محمول — معبأ** · ☐ **حبوب كافيين — معبأة** *اختيارية — لليقظة السريعة أثناء إنذارات الليل. تخطّاها إذا كنت حساساً للكافيين.* |
| **الإقامة الليلية** | ☐ **مستلزمات دينية — معبأة** *سيدور، تفيلين* · ☐ **طعام / شراب — معبأ** *للإقامات الطويلة* · ☐ **قناع عين وسدادات أذن — معبأة** *للنوم في الملجأ* · ☐ **زجاجات ماء** · ☐ **مظلات** · ☐ **راوتر محمول + إيثرنت + مصدر تيار متردد** · ☐ **محولات / موزعات كهرباء** · ☐ **خيمة / بطانية** |
| **وثائق السفر** | ☐ **جوازات السفر — معبأة ومُتحقق من صلاحيتها** · ☐ **نقود — متوفرة / معبأة** · ☐ **بطاقات الهوية — معبأة** · ☐ **وثائق السفر — معبأة** |
