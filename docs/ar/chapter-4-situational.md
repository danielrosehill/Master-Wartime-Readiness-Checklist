# الفصل الرابع: قوائم التحقق الظرفية

## 4.1 وضعية النهار في المنزل

*فحص سريع للحالة أثناء التواجد في المنزل نهاراً. شغّله بعد الاستيقاظ، وأعده بعد أي انقطاع.*

| | العنصر | التحقق |
|---|------|-------|
| 📱 | **الهاتف** ⚠️ | ☐ **معك شخصياً** *أو في متناول اليد* · ☐ **مشحون أو يشحن** · ☐ **تطبيق HFC يعمل** |
| 👕 | **الملابس** ⚠️ | ☐ **مرتدي ملابس كاملة** · ☐ **حذاء مغلق مرتدى** *ليس شباشب وليس حافي القدمين* |
| 🔑 | **المفاتيح** ⚠️ | ☐ **معك أو في/بجانب الباب** |
| 🎒 | **حقيبة الطوارئ** ⚠️ | ☐ **بجانب الباب** · ☐ **مغلقة** |
| 🚪 | **مسار الخروج** ⚠️ | ☐ **الممر خالٍ** · ☐ **الباب الأمامي يمكن فتحه بسرعة** |
| 💻 | **تنبيه المتصفح** | ☐ **إضافة الإنذار الأحمر نشطة** *إذا كنت تعمل على المكتب* |

!!! tip "العمل من المنزل أثناء الحرب"
    - ابقَ مرتدياً ملابسك وحذاءك — لا تتحول إلى البيجاما "فقط على المكتب"
    - سماعات الرأس: استخدم أذناً واحدة فقط، أو اجعل الصوت منخفضاً بما يكفي لسماع صفارة الإنذار
    - احفظ عملك بشكل متكرر — قد تحتاج لمغادرة مكتبك في منتصف جملة
    - في بداية أي مكالمة: *"أنا في منطقة تنبيه نشطة. قد أحتاج للمغادرة فجأة."*

## 4.2 بعد العودة من الملجأ (إعادة الضبط)

*شغّلها فوراً بعد إشارة الأمان. أعد الجاهزية الكاملة قبل التنبيه التالي.*

| | العنصر | التحقق |
|---|------|-------|
| 🎒 | **حقيبة الطوارئ** ⚠️ | ☐ **أعيدت بجانب الباب** · ☐ **أُعيد إغلاقها** *هل سقط شيء؟ أعده الآن* |
| 📱 | **الهاتف** ⚠️ | ☐ **بالشحن** *إذا انخفضت البطارية* · ☐ **تطبيق HFC لا يزال يعمل** |
| | **بنك الطاقة** | ☐ **بالشحن** *ربما استُخدم في الملجأ* |
| 💧 | **الماء** | ☐ **تم التجديد** *إذا شربت من المخزون الطارئ* |
| 👕 | **الملابس والأحذية** | ☐ **أُعيدت لمكانها** *بجانب السرير (ليلاً) أو مرتداها (نهاراً)* |
| | **المصباح** | ☐ **أُعيد لمكانه** *منضدة جانبية أو حقيبة الطوارئ* |
| 👥 | **المعالون** ⚠️ | ☐ **الجميع بخير وفي أمان** |

!!! note "العناية بالذات"
    **العناية الذاتية — افعل هذه الآن، ليس لاحقاً.** لا تعرف متى يأتي التنبيه التالي.

    - [ ] **أكلت** — *التوتر يقمع الشهية — كل على أي حال. تحتاج الطاقة.*
    - [ ] **شربت**
    - [ ] **تم** *المرحاض*
    - [ ] **استحممت** — *إذا سمح الوقت والوضع*
    - [ ] **تم التجميل** — *الروح المعنوية والشعور بالحياة الطبيعية مهمان*
    - [ ] **استرحت** — *عند الإمكان*

    **الأطفال:**

    - [ ] **أُطعموا** · **شربوا** · **ذهبوا للمرحاض / تم تغييرهم**
    - [ ] **نُظّفوا** *إن أمكن* · **استراحوا** *ضعهم للقيلولة إذا سمح الوقت*

## 4.3 قبل الاستحمام

| | العنصر | التحقق |
|---|------|-------|
| 👁 | **فحص الأخبار** ⚠️ | ☐ **تم مسح الوضع** *لا توجد تنبيهات نشطة في منطقتك* |
| 📱 | **الهاتف** ⚠️ | ☐ **الصوت على أعلى مستوى** *يجب أن يُسمع فوق صوت الماء* · ☐ **في الحمام أو خارج الباب مباشرة** |
| 👕 | **الملابس** ⚠️ | ☐ **طقم كامل في الحمام** *جاهز للارتداء فوراً — ليس في غرفة أخرى* |
| 👟 | **الأحذية** ⚠️ | ☐ **حذاء مغلق بجانب باب الحمام** |
| | **المنشفة** | ☐ **في متناول اليد** *للتجفيف السريع* |
| 🚪 | **الباب** | ☐ **باب الحمام غير مقفل** |
| ⏰ | **اجعله قصيراً** ⚠️ | ☐ **الوقت مُقلّص** *أنت في أضعف حالاتك؛ كن سريعاً* |

## 4.4 قبل النوم

| | العنصر | التحقق |
|---|------|-------|
| 👁 | **فحص الأخبار** | ☐ **تم مسح الوضع** *قيّم ما إذا كان آمناً للنوم في المنزل* |
| 📱 | **الهاتف** ⚠️ | ☐ **يشحن أو مشحون** · ☐ **ليس في وضع الطيران** *وضع عدم الإزعاج/الصامت مقبول* · ☐ **خدمات الموقع مُفعّلة** · ☐ **تطبيق HFC — صلاحيات الإشعارات والتجاوز** |
| 👕 | **الملابس** ⚠️ | ☐ **موضوعة بجانب السرير** *طقم كامل جاهز في ثوانٍ* |
| 👟 | **الأحذية** ⚠️ | ☐ **حذاء مغلق بجانب السرير** *جاهز للارتداء فوراً* |
| | **المصباح** | ☐ **في متناول اليد** *على المنضدة الجانبية، ليس مدفوناً في حقيبة الطوارئ* |
| | **النظارات** | ☐ **بجانب السرير** *نفس المكان كل ليلة* |
| | **حبة كافيين *(اختيارية)*** | ☐ **بجانب السرير مع ماء** *قد ترغب في تناول واحدة عند الاستيقاظ على صفارة إنذار لتسريع اليقظة. غير مناسبة لمن يعانون من أمراض قلبية أو اضطرابات قلق أو حساسية للكافيين. استشر طبيبك.* |
| 🔑 | **المفاتيح** | ☐ **يمكن الوصول إليها** *بجانب السرير أو على خطاف بجانب الباب الأمامي* |
| 🎒 | **حقيبة الطوارئ** ⚠️ | ☐ **بجانب الباب** · ☐ **المحتويات مُتحقق منها** · ☐ **بنك الطاقة يشحن** |
| 🚪 | **مسار الخروج** ⚠️ | ☐ **الممر خالٍ** *يمكن التنقل في الظلام* · ☐ **الباب الأمامي ليس مقفلاً بقفل مزدوج** |
| 📡 | **السمع** ⚠️ | ☐ **بدون سدادات أذن** *يجب أن تسمع صفارة الإنذار وتنبيه الهاتف* |
| 👶 | **المعالون** ⚠️ | ☐ **الجميع في مكانهم** · ☐ **الطفل — مُلبّس أو ملابس بجانب السرير، حمّالة الطفل بجانب الباب** |
