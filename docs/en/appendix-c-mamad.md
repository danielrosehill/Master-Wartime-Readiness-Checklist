# Appendix C: Mamad (Protected Space) Inspection

*Based on Home Front Command guidelines for residential protected spaces.*

### Structural Inspection

| ITEM | CHECK |
|------|-------|
| **Blast Door** | ☐ **OPENS AND CLOSES EASILY** · ☐ **HANDLE TURNS 90° UPWARD** · ☐ **DOUBLE LOCK FUNCTIONS** |
| **Door Seal** | ☐ **RUBBER INSULATION PRESENT** *Not dried out or cracked — replace if brittle* |
| **Light Test** | ☐ **NO LIGHT PENETRATION WHEN SEALED** *Light = inadequate seal* |
| **Steel Outer Window** | ☐ **OPENS AND CLOSES EASILY** |
| **Glass Inner Window** | ☐ **FUNCTIONS CORRECTLY** · *Double-wing: remove both wings during escalation and store outside room* |
| **Ventilation Pipes** | ☐ **RUBBER INSULATION INTACT** · ☐ **STEEL COVER SCREWS TIGHTEN FULLY** |

### Contents & Safety

| ITEM | CHECK |
|------|-------|
| **Flammable / Hazardous** | ☐ **NONE** *No flammable, hazardous, or toxic materials* |
| **Glass / Ceramics** | ☐ **NONE** *No glass, ceramic, porcelain that could shatter* |
| **Heavy Items** | ☐ **SHELVES AND HEAVY ITEMS FIXED TO WALLS** |
| **Gas Tanks** | ☐ **3+ METRES FROM PROTECTED ROOM WALLS** |
| **Emergency Supplies** | ☐ **STORED INSIDE OR IMMEDIATELY NEARBY** |
| **Ceiling Fan** | ☐ **NONE INSTALLED** *Prohibited in mamads* |
| **Ventilation** | ☐ **ROOM IS VENTILATED IF USED AS BEDROOM** · *Must NOT be used as kitchen, bathroom, or washroom* |
