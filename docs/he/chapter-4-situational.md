# פרק 4: רשימות מצביות

## 4.1 מצב יום בבית

*בדיקת מצב מהירה בזמן שהייה בבית במהלך היום. בצעו לאחר ההשכמה, וחזרו לאחר כל הפרעה.*

| | פריט | בדיקה |
|---|------|-------|
| 📱 | **טלפון** ⚠️ | ☐ **עליך** *או במרחק יד* · ☐ **טעון או בטעינה** · ☐ **אפליקציית פיקוד העורף פעילה** |
| 👕 | **לבוש** ⚠️ | ☐ **לבוש במלואו** · ☐ **נעליים סגורות** *לא כפכפים, לא יחף* |
| 🔑 | **מפתחות** ⚠️ | ☐ **עליך או בדלת / ליד הדלת** |
| 🎒 | **תיק חירום** ⚠️ | ☐ **ליד הדלת** · ☐ **סגור** |
| 🚪 | **מסלול יציאה** ⚠️ | ☐ **מסדרון פנוי** · ☐ **דלת כניסה ניתנת לפתיחה מהירה** |
| 💻 | **התרעת דפדפן** | ☐ **תוסף צבע אדום פעיל** *אם עובדים ליד המחשב* |

!!! tip "עבודה מהבית בשעת מלחמה"
    - הישארו לבושים ועם נעליים — אל תחליפו לפיג׳מה "רק ליד השולחן"
    - אוזניות: השתמשו באוזנייה אחת בלבד, או שמרו על עוצמה נמוכה מספיק כדי לשמוע צופר
    - שמרו עבודה לעיתים קרובות — ייתכן שתצטרכו לעזוב את השולחן באמצע משפט
    - בתחילת כל שיחה: *"אני באזור התרעה פעיל. ייתכן שאצטרך לעזוב בפתאומיות."*

## 4.2 לאחר חזרה מהמקלט (איפוס)

*בצעו מיד לאחר ה"ניתן לצאת". שחזרו מוכנות מלאה לפני ההתרעה הבאה.*

| | פריט | בדיקה |
|---|------|-------|
| 🎒 | **תיק חירום** ⚠️ | ☐ **חזר ליד הדלת** · ☐ **נסגר מחדש** *משהו נפל? החזירו עכשיו* |
| 📱 | **טלפון** ⚠️ | ☐ **בטעינה** *אם הסוללה ירדה* · ☐ **אפליקציית פיקוד העורף עדיין פעילה** |
| | **מטען נייד** | ☐ **בטעינה** *ייתכן שנעשה בו שימוש במקלט* |
| 💧 | **מים** | ☐ **מלאי הושלם** *אם שתיתם ממלאי החירום* |
| 👕 | **בגדים ונעליים** | ☐ **חזרו למקום** *ליד המיטה (לילה) או על הגוף (יום)* |
| | **פנס** | ☐ **חזר למקום** *שידה או תיק חירום* |
| 👥 | **תלויים** ⚠️ | ☐ **כולם ידועים ובטוחים** |

!!! note "רווחה"
    **טיפול עצמי — עשו את זה עכשיו, לא אחר כך.** אתם לא יודעים מתי תגיע ההתרעה הבאה.

    - [ ] **אכלתי** — *לחץ מדכא תיאבון — אכלו בכל זאת. אתם צריכים את האנרגיה.*
    - [ ] **שתיתי**
    - [ ] **שירותים — בוצע**
    - [ ] **התקלחתי** — *אם הזמן והמצב מאפשרים*
    - [ ] **טופחתי** — *מורל ותחושת שגרה חשובים*
    - [ ] **נחתי** — *כשאפשר*

    **ילדים:**

    - [ ] **אכלו** · **שתו מים** · **שירותים / החלפה**
    - [ ] **ניקיון** *אם אפשר* · **מנוחה** *הניחו לשנת צהריים אם הזמן מאפשר*

| 🚪 | **מסלול יציאה** ⚠️ | ☐ **מסדרון פנוי** *ניתן לנווט בחושך* · ☐ **דלת כניסה לא נעולה בנעילה כפולה** |

| 👶 | **תלויים** ⚠️ | ☐ **כולם ידועים** · ☐ **תינוק — לבוש או בגדים ליד המיטה, נשאית ליד הדלת** |
